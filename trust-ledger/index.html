Trust Ledger

Immutable compliance log for pre-execution governance

Trust Ledger is the audit lineage brand for Noetfield decisions: every evaluate, veto, and policy reference tied to a Request ID (RID) and exportable for boards, OSFI E-23 vendor packs, and partner due diligence. It is not a payment ledger, wallet, or custody product.

How it is produced

  1. Evaluate — policy decision with reason_code and policy_refs tied to a Request ID
  2. Record — entries append to the governed audit log (integrity hash per event)
  3. Export — audit-export by RID for procurement (Governance API reference)
