## Normalized purpose

This document focuses on practical tool-stack and workflow guidance for
AI-heavy development on MacBook M5 Pro, agentic coding, multi-file refactoring,
system architecture, startup velocity, and hybrid local/cloud orchestration.

## Key guidance

- The IDE is no longer the center; context layer and persistent agent runtime
  are central.
- Use Cursor as primary AI-native execution surface.
- Use Claude Code for deep reasoning and architecture.
- Use GitHub Copilot for reliable inline production coding.
- Use Continue.dev for programmable multi-model routing.
- Use local models for speed/privacy and cloud models for deep reasoning.
- Developer role evolves toward governor/orchestrator of multi-agent systems.

## Registry classification

Classify as an active reference for developer workflow and compute/tooling
strategy. It is not an active governance rule and does not supersede the
