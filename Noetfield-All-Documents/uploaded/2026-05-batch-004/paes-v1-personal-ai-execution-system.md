# PAES v1.0 - Personal AI Execution System

Document key: `paes-v1-personal-ai-execution-system`

Source status: always-on digital twin and autonomous operator layer

## Normalized purpose

This document defines a three-layer autonomous personal AI system:

- Cognitive Twin
- Execution Agent
- Telegram Control Interface

It frames the system as a persistent AI operator that simulates user behavior,
executes real-world tasks across devices, and reports through Telegram.

## Key architecture

- Telegram Layer
- Orchestration Layer
- Cognitive Core
- Tool and Execution Layer
- Device Bridge Layer
