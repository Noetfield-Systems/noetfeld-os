## Normalized purpose

This document defines the Security Agent as the trust boundary of the
Noetfield Developer OS.

The Security Agent is not merely a scanner. It is a governed system service for
multi-agent and capital-aware operations.

## Responsibilities

- static code analysis
- dynamic analysis
- context sensitivity enforcement
- capital security
- agent behavior monitoring
- policy enforcement
- risk reports
- recommended fixes

Classify as active reference for the future Agent Behavior Monitoring and
Security Agent work packages. It is not yet executable policy until translated
